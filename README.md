# ReconForge

ReconForge ist ein modulares Reconnaissance-
und Bug-Bounty-Tool.

Das Projekt wurde im Rahmen des Moduls
Softwaretechnik entwickelt.

## Funktionen

- Domain Scan
- Subdomain Discovery
- IP Resolution
- Port Scan
- JSON Export
- CLI Interface

## Projektziele

Das Ziel des Projekts ist die Entwicklung
eines modularen und erweiterbaren
Security-Tools für Reconnaissance-Prozesse.

## Technologien

- C#
- .NET
- Clean Architecture
- OOP
- Unit Testing

## Verwendete Werkzeuge

- Notion
- Jira
- GitHub
