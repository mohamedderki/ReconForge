# ReconForge Mission

ReconForge ist ein modulares Bug-Bounty-
und Reconnaissance-Tool.

Das Projekt unterstützt Security Research
und automatisierte Recon-Workflows.

Das Tool analysiert Domains,
Subdomains, IP-Adressen
und Netzwerkdienste.

ReconForge soll einfach erweiterbar
und leicht wartbar sein.

Die Entwicklung basiert auf
Clean Architecture,
Object-Oriented Programming
und Softwaretestung.
